งานนี้คือโปรเจกต์ส่งท้ายสัปดาห์แรกของการฝึกอบรมเว็บครับ โจทย์คือให้คุณสร้าง "Mini one-page site" (เว็บไซต์หน้าเดียวขนาดเล็ก) เพื่อรวบรวมพอร์ตโฟลิโอของคุณเอง โดยมีรายละเอียดดังนี้ครับ: 🎯 เป้าหมายหลัก สร้างหน้าเว็บ 1 หน้า ที่เป็นหน้าสรุปและมีลิงก์เชื่อมไปยัง CV (เรซูเม่) 2 อัน ของคุณ โดยเน้นความ Responsive (เปิดดูในมือถือหรือคอมพิวเตอร์ก็สวย) และแนะนำให้ใช้ Bootstrap เข้ามาช่วย 📝 สิ่งที่ต้องมีในหน้าเว็บ (4 ส่วนหลัก) Navigation bar (แถบเมนู): ต้องตั้งค่าให้ ติดอยู่กับที่ (Sticky/Fixed) เสมอแม้จะเลื่อนหน้าจอลงมา เพื่อให้กดไปส่วนต่างๆ ได้ง่าย About section (ส่วนแนะนำตัว): ใส่ชื่อ, รูปถ่ายตัวเอง และประวัติย่อๆ เช่น การศึกษา หรือความสนใจส่วนตัว Showcase section (ส่วนแสดงผลงาน): ประสบการณ์ทำงาน, ทักษะ (Skills) หรือความสำเร็จต่างๆ ที่ผ่านมา Contact section (ส่วนติดต่อ): ใส่ลิงก์โซเชียลมีเดียหรือโปรไฟล์งาน เช่น Instagram, GitHub, หรือ LinkedIn 💡 คำแนะนำในการทำ (Step-by-Step) 1. วางโครงสร้างด้วย HTML ใช้ Tag พื้นฐานแบ่งส่วนตามโจทย์:

สำหรับช่องทางติดต่อ 2. ตกแต่งด้วย CSS (หรือ Bootstrap) ถ้าใช้ Bootstrap (แนะนำ): จะช่วยให้ทำ Responsive ง่ายมาก แค่ใส่ Class เช่น container, row, col และใช้ Class fixed-top สำหรับ Navigation bar ถ้าเขียน CSS เอง: อย่าลืมใช้ position: fixed; สำหรับเมนู และ flexbox หรือ grid ในการจัดวางเลย์เอาต์ 3. ทำให้เป็น One-page ใน Navigation bar ให้ตั้งค่าลิงก์ href="#id-ของส่วนนั้นๆ" (เช่น Contact) เพื่อให้พอกดแล้วหน้าจอเลื่อนลงไปส่วนนั้นทันที 4. ใส่ลิงก์ CV อย่าลืมทำปุ่มหรือข้อความที่คลิกแล้วเปิดไฟล์ CV ของคุณได้ (อาจจะเป็นไฟล์ PDF หรือลิงก์ Google Drive) เครื่องมือแนะนำ: Code Editor: VS Code Template: โจทย์บอกว่าอนุญาตให้ใช้ Template ได้ คุณสามารถหา "Free Bootstrap Portfolio Template" มาปรับแก้ (Customize) ตามข้อมูลตัวเองได้เลยครับ จะช่วยให้งานเสร็จไวและสวยงามตามที่โจทย์ต้องการ!
